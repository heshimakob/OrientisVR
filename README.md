## Prerequisites

- PHP >= 5.3.7
- NPM >= 6.13.4
- Node >= 12.22.1

## Installations

```bash

# Clone the repository (stable branch)
git clone -b master https://github.com/heshimakob/Orientis.git
# OR download code

npm install

```

### Dev mode (hot-code reloading, error reporting, etc.)

```bash
quasar dev
```
